# Bus Ticket Booking

This project is a web-based bus ticket booking application. It allows users to search for buses, select seats, provide passenger information, and review their bookings. The application is built using React, Redux Toolkit, and Tailwind CSS, and is powered by Vite for development and build processes.

## Features

- Search for buses between cities.
- Filter buses based on travel date.
- Select seats and provide passenger details.
- Review and confirm bookings.
- View ticket details.

## Prerequisites

- Node.js

## Getting Started

Follow these steps to set up and run the project locally:

1. **Clone the repository:**

   git clone <repository-url>
   cd bus-ticket-booking

2. **Install dependencies:**

   npm install

3. **Run the development server:**

   npm run dev -for local

   Use this Url to access the Application 'http://localhost:5173'.
